# 👁️ Project Mimic

> **A multiplayer psychological horror game built with Three.js and TypeScript.**

---

## 🎮 About

Project Mimic is a multiplayer horror experience where players explore a quiet neighborhood while being hunted by an unknown entity capable of perfectly imitating human behavior.

Nothing is ever quite what it seems.

Search buildings, uncover the truth, survive... or become part of it.

---

## ✨ Features

- 🏃 First Person Gameplay
- 👥 Multiplayer
- 👁️ Advanced Mimic AI
- 🏠 Fully Explorable Buildings
- 🔦 Flashlight System
- 🎒 Inventory System
- 📖 Story Driven
- 🎭 Multiple Endings
- 🎵 Dynamic Horror Audio
- 💡 Atmospheric Lighting

---

## 🛠️ Built With

- TypeScript
- Three.js
- Vite
- HTML5
- CSS3
